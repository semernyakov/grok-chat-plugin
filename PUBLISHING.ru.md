# Руководство пользователя

## Основные функции

### Чат с ИИ

1. Вызовите чат командой `/groq`
2. Введите ваш запрос
3. Получите ответ от выбранной модели
4. Продолжайте диалог в контексте

### Работа с кодом

- Подсветка синтаксиса для всех популярных языков
- Автодополнение кода
- Форматирование кода
- Объяснение кода
- Рефакторинг

### Форматирование текста

- Полная поддержка Markdown
- Таблицы
- Списки
- Цитаты
- Математические формулы

### История и контекст

- Сохранение истории диалогов
- Контекстно-зависимые ответы
- Экспорт истории
- Поиск по истории

## Модели и их применение

### Llama 3 70B Versatile
- Общие задачи
- Сложные рассуждения
- Творческое письмо
- Анализ текста

### Llama 3 8B Instant
- Быстрые ответы
- Простые задачи
- Базовые вопросы

### Mixtral 8x7B
- Многоязычные задачи
- Генерация кода
- Техническая документация

### Специализированные модели
- Модерация контента
- Транскрипция аудио
- Анализ изображений (бета)

## Советы и приёмы

### Оптимизация запросов
1. Будьте конкретны
2. Используйте контекст
3. Указывайте формат ответа
4. Разбивайте сложные задачи

### Работа с контекстом
- Используйте предыдущие ответы
- Ссылайтесь на части диалога
- Уточняйте детали

### Экономия токенов
- Избегайте повторений
- Используйте краткие формулировки
- Выбирайте подходящую модель

## Безопасность

### Конфиденциальность
- Не отправляйте личные данные
- Используйте шифрование
- Регулярно меняйте API ключ

### Ограничения
- Соблюдайте лимиты запросов
- Следите за использованием токенов
- Учитывайте размер контекста

## Обновления и поддержка

### Обновление плагина
1. Проверяйте наличие обновлений
2. Читайте список изменений
3. Делайте резервные копии

### Получение помощи
- GitHub Issues
- Обсуждения сообщества
- Документация
